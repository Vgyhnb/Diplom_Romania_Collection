We live in a time when consumers are increasingly relying on their mobile devices to make purchases. In this application, users can view a wide range of furniture, select products in various ways using analytical tools of artificial intelligence, visualize pieces of furniture in their own interior thanks to augmented reality technology. The app will help consumers make quick decisions when choosing furniture and enjoy shopping.

01 /Task

  • Task

    To make a convenient and functional application that will allow customers to easily and quickly select furniture for the home, and the online store Romania Collection to distinguish among competitors, increase conversions and sales.

  • Solution

    Development of an application with a convenient modern interface for searching, filtering and comparing products. The use of artificial intelligence and augmented reality technologies that will help users explore the store's assortment and offer suitable products. Development of modern and stylish design.

02 / Working process

  • Idea

    Development of a mobile application with the latest technologies.

  • Study

    Analysis of information and target audience. Studying the needs of buyers.

  • UX Analysis

    Studying the user experience. Development of information architecture.

  • Wireframes

    Preparation of UX and UI mudboards. Prototyping an application for iPhone 14 Pro phone screens.

  • Design

    Application design development.

    06 / UI design

    Screensaver

    The first thing the user sees on the screen is the screensaver. She draws attention to the store's assortment – "Elegant Home furniture".

    Registration

    The application has a convenient and fast registration using social networks, phone and e-mail. The user has the option to skip or continue registration.

    Onboarding

    The onboarding screen represents a mobile application, its navigation system, functions and benefits.

    Personal account

    In the personal account, the user can view the order history, enter their payment details, delivery address, check the store's notifications, find answers to frequently asked questions and view the store's contacts.

    Main Screen

    This is the main screen through which users interact with most of the application options. On the main screen there is a quick search, popular products, categories, collections. The user-friendly interface allows users to easily find the furniture they need.
